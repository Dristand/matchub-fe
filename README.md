# Club Application - Frontend

## Description

Frontend of Club Application app.

## Requirements
1. Git (Latest)
2. NPM

```bash
$ git clone https://github.com/Dristand/matchub-fe.git
$ cd matchub-fe
$ npm install
$ npm run dev
```

## Important URL
```bash
# Normal URL
$ host:3000/

# If you have the backend on the same machine
$ host:3001/
```
